# Change Log

## [0.0.1] - 2025-12-26

### Added
- Multi-threaded call stack visualization in a merged graph.
- Thread grouping logic to unify identical execution paths.
- Active frame highlighting (editor gutter arrow + graph node highlight).
- Manual thread splitting/merging features via tooltip.
- Path/Branch highlighting on hover for easier stack tracing.
- Automatic navigation to source code on node selection.
- Custom thread labels (ID and name) above leaf nodes.
